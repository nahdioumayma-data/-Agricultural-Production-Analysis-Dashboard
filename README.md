
 🌾 Agricultural Production Analysis Dashboard

## 📋 Description
Ce projet Power BI propose une analyse multidimensionnelle de la production agricole. L'objectif est d'aider les exploitants et décideurs à comprendre l'impact des variables environnementales et techniques sur les rendements pour optimiser les récoltes.

Le dashboard s'articule autour de trois axes stratégiques :
1.  **Vue Globale :** Suivi des KPIs principaux (Rendement total, moyenne, durée de croissance).
2.  **Climat & Pratiques :** Analyse de l'impact de l'irrigation, de la fertilisation et de la météo.
3.  **Cultures & Sols :** Corrélation entre les types de sols (Sandy, Clay, Loam, etc.) et les variétés de cultures.

## 🚀 Fonctionnalités Clés
* **Analyse Comparative :** Mesure de l'écart de performance entre cultures irriguées ($5,25$ t/ha) et non-irriguées ($4,05$ t/ha).
* **Segmentation Géographique :** Filtres par régions (East, North, South, West) pour une analyse locale.
* **Visualisations Avancées :** Utilisation de Treemaps pour la répartition des cultures et de graphiques en anneau pour les types de sols.
* **Interactivité :** Navigation fluide entre les pages via un menu de boutons intégré.

## 🛠️ Technologies Utilisées
* **Power BI Desktop** : Modélisation des données et création des visuels.
* **DAX (Data Analysis Expressions)** : Création de mesures calculées pour les rendements et moyennes.
* **Power Query** : Nettoyage et transformation des données (ETL).

## 📊 Aperçu des Données
* **Rendement Total :** 4,65M de tonnes.
* **Cycle de récolte moyen :** 104,5 jours.
* **Température moyenne :** 27,5°C.
